Cpp小技巧用法积累--长期更新

发表于 2019-10-17 更新于 2019-11-11 分类于 编程 阅读次数： Valine：
本文字数： 374 阅读时长 ≈ 1 分钟
刚接触C++，对很多东西都非常陌生，也是跳过了C直接学习的C++。这里会长期更新一些C/C++的小知识点和使用技巧。
  • static_cast, reinterpret_cast, dynamic_cast

static_cast 用法：static_cast < type-id > ( expression ) , 把expression转换成type-id，属于显视强制转换

reinterpret_cast 主要将数据从一种类型转换为另一种类型， 所谓“通常为操作数的位模式提供较低层的重新解释”也就是说将数据以二进制存在形式的重新解释。

其实就是将指正指向的类型重新转换了，比如之前指正指向的是int类型，可转换为char型，指向的对象还是那些对象，只不过现在需要重新安装新的类型进行“编译”了。可参照以下两个链接，解释的非常好：
