波束形成技术（Beamforming）

Posted on 2019-11-06 Edited on 2019-11-08

波束形成技术（Beamforming）

​ 近年来，语音识别带起了前端信号处理的小高潮，经由前端处理之后馈送给识别引擎信号的质量（信噪比、语音保真度等）对识别率至关重要。远场、混响、噪声等不利于语音质量的因素等在前端信号处理之后，都会在某种程度上得到削弱，同时这也意味着语音信号会在某种程度上得到增强。

​ 语音增强，总体来说有三种方法，即在三个不同域上的操作，比如频域上的谱减法降噪，时域上的自适应滤波器做回声消除，以及本项目研究的空域上的波束形成（Beamforming）。波束形成可以看作是空间滤波器，其许多原理都可以类比经典数字信号处理中的滤波器（混叠、加窗）。这很好理解，毕竟都是采样，只不过一个是对空间，一个是对时间罢了。波束形成的形状，指向，旁瓣的大小都是可设计的。本文档即基于Python进行模拟仿真一些基础常见的波束形成技术，包括频域（Delay and Sum/ MaxDF等）及时域的波束形成技术。

Python环境为Python3.6，需要的包有numpy/scipy/matplotlib, 参考文献主要为:

[1] J.Benesty, I.Cohen, and J.Chen, Fundamentals of Signal Enhancement and Array Signal Processing. Weily-IEEE Press 2017.

[2] Nuance 内部培训材料

一、声场统计模型（Acoustic Model）

声传播模型

​ 声传播的过程，即波动的过程，是一个涉及到空间、时间的物理过程。根据波阵面的不同，可以简单的分为球面波和平面波（还有线型声源扩散，波阵面为圆柱面）。石头落入水中荡起的涟漪好比是球面波的横截面，能量不停的往外扩散，其波阵面是球形。而平面波则更好理解了。两张图，下面左侧为球面波的截面，右侧为平面波的截面。

​ 实际上我们身处的声学环境，大部分都可假设是球面波，即可认为发声源是一点源。从上面左图可以看到，当距离声源足够远时（左图两个黑点），圆弧近似看作为直线，此时认为左侧的场景同右侧一样，是为平面波。这就好比我们站在地球上，感觉不到地球是圆的一样，是同一个道理。以上就是平面波假设，是我们研究阵列及其他一些算法的重要前提假设。

声场统计模型

​ 假设声场中任一点的信号是一个时空域的随机采样过程。描述声场中任意两点不同时刻信号之间的相关系数为两点信号相乘的期望，可以写作为：

$r_{y_i,y_l}(r_i,r_l) = E[Y(r_i,t_0,\eta)Y(r_l,t_1,\eta)]$

而在匀强声场中，上述则可改写为：

$r_{y_i,y_l}(d, \tau) = E[Y(r_i,t_0,\eta)Y(r_i+d,t_1+\tau,\eta)]$

注意上述是一个时空域的概念（要区别于下面的功率谱）。在研究声场时，更一般的假设则是：静态且各向同性的，因此上述距离d可以加个绝对值。声场的空间功率谱密度，借鉴随机信号的功率谱密度，由维纳钦金定理得到，信号的功率度密度等于信号自相关系数的傅立叶变换：

$\Phi_{y_i,y_l}(\omega) = \int_{-\infty}^{+\infty} r_{y_i,y_l}(d,\tau)exp(-j\omega\tau)d\tau$

归一化的空间功率谱（Normalized Spatial Cross-PSD）写作：

$J(\omega) = \frac{\Phi_{y_i,y_l(\omega)}}{\sqrt{\Phi_{y_i,y_i}(\omega)\Phi_{y_l,y_l}(\omega)}}$

上式子也即幅度均方相干系数（Magnitude Squared Coherence，MSC），用以描述空间声场的相干性，主要是频域的概念：

1.当MSC = 1时，声场为Coherence Sound，完全相干声场

2.当MSC = 0时， 声场为Incoherent Sound，完全不相干声场，比如麦克风的底噪、电流噪音

一类比较特殊的声场——扩散场（能量分布均匀，各个方向作无规分布的声场），其MSC：

$J_\Theta(\omega):= \frac{1}{4\pi}\int^{2\pi}_{0}\int_0^{2\pi}exp(-j\omega\tau\cos{\theta})\sin{\theta}d\theta d\phi
= \frac{\sin(\omega\tau)}{\omega\tau}$

这在后面研究在不同声场下麦克风的表现都会用得到。

上面是扩散声场中，不同距离之间的MSC变化图，可以看到

1.低频时，MSC近似为1，可以认为是相干声场，高频时MSC接近于0，可以认为是不相干声场。

2.两点之间的距离是MSC的重要决定参数，这个是肯定的，几乎所有于频率有关的参数，都要牵涉到距离。

3.车内噪声场、后期混响场可以认为是典型的扩散场。

其他的类型的声场（混合声场），都可以用上述模型进行加权逼近。

房间声学

​ 关于这部分，我其实算是科班出身，我研究生主攻建筑声学，其中一门重要的课程就是Kutruff的《Room Acoustics》，授课老师即是鄙人之导师，所以备课预习格外认真，且老师授课方式很新颖，教学内容印象深刻，受益匪浅。所以，下面也不仔细说了，流水账翻一翻吧。

​ 上图，左侧小人是声源，可以认为是喇叭，右边的小人是听众，可以认为是麦克风，麦克风接受到的信号除了直达声，还有经过墙面一次反射延后到达的一次反射声，经过墙面二次反射延后到达的二次反射声。。

墙面反射一次（或者传播途径远），可以认为声波能量产生了一定比列的损失（乘以损失系数），延后到达即认为产生一次时延，假设声源发出的信号为 $x(t)$ 接收到的信号则为：

$y(t) = x(t) + a_1x(t-t_0) + a_2x(t-t_1)+ a_3x(t-t_3)+…..$

上式是不是和滤波器的差分方程很像？！没错，这就是房间的脉冲响应（Impulse Response），是房间传递函数在时域上的表现形式。别小瞧这个式子，时域卷积运算的时候为什么反转序列移序相乘再相加，参透上式即可得彀中玄妙。

二、固定波束形成技术（FixedBeaming）

​ 其实波束形成的玩法本质上就是操作阵列不同麦克风上的信号加权系数，对于一类固定滤波器系数的阵列来说，无论输入信号或者噪声信号是啥样的（统计特征），其滤波器系数雷打不动，此类波束形成叫Fixed Beamforming，这好比传统数字信号处理里面的经典滤波器；可以联想到，现代数字信号里面有一类自适应滤波器，对应的，阵列里就有相应的Adaptive Beamforming，其滤波器的系数是跟随着噪声场的变化而变化的。

​ 此外，阵列根据形状不同还分为线阵、环针、中心阵等，为了便于理解，我们先从一维的Fixed Beamforming开始着手研究uniform linear arrays，由此情况推导其他以及引申到普适情况也不会是什么难事。

1. 问题建模及求解

​ 远场假设，一声源与共有M个麦克风线阵的轴线夹角呈$\theta_d$入射，麦克风间距为 $\delta$, 则每两个麦克风之间的单位时延为 $\tau_0 = cos(\theta_d)*\delta/c$ ，其中c为声速，则以最右侧，也即离声源最近的麦克风为参考可以定义一个导向矢量(Steering Vector)：

$d(f, \cos(\theta)) = [1 e^{-j2\pi f\tau_{0}\cos(\theta)} … e^{-j(M-1)2\pi f\tau_{0}\cos(\theta)}]^T$

​ 根据傅立叶变换的性质，时域左移或者右移动等于频域在相位上的改变。根据下式可以看出导向矢量是对频域相移的补偿。比如第一个元素是1，那么不需要任何补偿，第二个元素，波阵面延后到达，收到的信号应该应该是乘以一个 $e$ 指数为正的相移因子，导向矢量补偿因子之后使得麦克风2收到的信号与麦克风1收到的信号相位上是对齐的，便于后面进一步的加权操作。如果现在没有对麦克风信号进行任何的系数加权，则收到的裸信号为：

$y(f) = [Y_{1}(f) Y_{2}(f) … Y_{M}(f)]^T =x(f)+v(f)
=d(f, \cos(\theta_d))X(f) + v(f)$

其中 $v(f)$为噪声信号，上述前半部分是我们的desired signal，进而接受信号的功率谱密度:

$\Phi_{y}(f)= E[y(f)y_{H}(f)]
= \phi_{X}(f)d(f,\cos(\theta_d))d^{H}(f,\cos(\theta_d)) + \Phi_{V}(f)$

其中 $\phi_{X}(f)$ 为信号的方差，$\Phi_{V}(f)$ 为噪声的相关系数矩阵。

Fixed Beamforming过程中我们最终求解都是为了降低其他方向的噪声而尽可能不失真地提取到目标信号，操作核心思想是对接受信号 $y(f)$进行系数加权求和。

2. 线阵模型

上一部分最后提到加权求和，这就有点滤波器的意思了，时域上的滤波器就对过去的输入或者输出进行加权求和。而麦克风阵列呢，就是对声场中不同的空间点信号进行加权求和。如下：

$Z(f) = \sum_{m=1}^{M}H_m^{*}(f)Y_{m}(f) = h^{H}(f)y(f) = X_{fd}(f) + V_{rn}(f)$

$\Z(f)$ 是最终输出信号，$h(f) = [H_{1}(f) H_{2}(f) … H_{M}(f)]^T$ 是M个麦克风的加权系数，每个麦克风对应一个。$X_{fd}(f) = X(f)h^{H}(f)d(f,\cos(\theta_d))$ 是我们的desired 信号，要求做到不失真满足 $h^{H}(f)d(f,\cos(\theta_d))=1$ 即可。记住这个式子，对后面求解的简化非常重要。 $V_{rn}(f) = h^{H}(f)v(f)$ 则是剩余的残留信号。

3.性能指标

评价麦克风阵列的性能指标无非是考察麦克风在不同声场下的表现，直接可以想到的有信噪比提升和指向性因数等。

  • 波束模式（beam pattern），理解为阵列在不同方位空间上的响应，其平方的形式上power pattern常常通过Polar画出。公式为:

$\begin{aligned} \mathcal{B}[\mathbf{h}(f), \cos \theta] &=\mathbf{d}^{H}(f, \cos \theta) \mathbf{h}(f) =\sum_{m=1}^{M} H_{m}(f) e^{\jmath(m-1) 2 \pi f \tau_{0} \cos \theta} \end{aligned}$

  • 阵列增益，阵列的输出信噪比比阵列的输入信噪比，推导过程不上了，公式为：

$\begin{aligned} \mathcal{G}[\mathbf{h}(f)] &=\frac{\operatorname{oSNR}[\mathbf{h}(f)]}{\operatorname{iSNR}(f)} =\frac{\left|\mathbf{h}^{H}(f) \mathbf{d}\left(f, \cos \theta_{\mathrm{d}}\right)\right|^{2}}{\mathbf{h}^{H}(f) \mathbf{\Gamma}_{\mathbf{v}}(f) \mathbf{h}(f)} \end{aligned}$

注意在Fixed Beamforming 中上述式子的分子为1。增益越大，说明麦克风对提升信噪比越有帮助。即理论上是越大越好，同理下面的白噪声增益（White Noise Gain）和指向性因数（Directivity Index）。

  • 当考察的噪声是白噪声时，空间中不同的任意两点的相关系数为0，所以上述 $\Gamma_V$ 是主对角为1，其余全为0 的单位阵。上述Array Gain可以写成：

$WNG = \frac{oSNR(f)}{iSNR(f)} = \frac{|h^{H}(f)d(f,\cos(\theta_d)|^2}{h^{H}(f) (f)h(f)}$

可以证明WNG数值上可达到最大值为M，即阵列中麦克风的个数。实际计算时在取对数。

  • 指向性因Directivity Factor（DI），则是考虑阵列工作于扩散声场，因此可同理写为：

    $DI = \frac{oSNR(f)}{iSNR(f)} = \frac{|h^{H}(f)d(f,\cos(\theta_d)|^2}{h^{H}(f) \Gamma_{0,\pi}(f)h(f)}$， 其中 $\Gamma_{0,\pi} =\frac {\sin(\omega\tau)}{\omega\tau}$翻看讲义一里面的声场统计模型，有关于此的讨论。

所以，关于任意beamforming性能指标的讨论，只要得到阵列的滤波器系数，其他（三指标）的自然都迎刃而解了。而许多beamforming技术也都是在这三个指标上玩花样，比如说最大化WNG或者DF。

3.空间混叠

开篇已经提及了，这里面的采样混叠其实和数字信号采样混叠道理是一样一样的，这里作一下推导演绎。

空间发生混叠，即阵列分不清信号来自哪个方向了，也就是存在两个不同的 $\theta_d$, 满足 $d(f,\cos(\theta_1)) = d(f,cos(\theta_2))$ , 观察steering vector，可以得到：

$\cos(\theta_1) = \frac{c}{fd} + \cos(\theta_2) =
\frac{\lambda}{d} + \cos(\theta_2)$ 或者 $\frac{\lambda}{d} = \frac{1}{\cos(\theta_1)-\cos(\theta_2)}$，此时

steering vector中的任一项是满足在两个角度下相等的，所以，此时发生了混叠。由基本三角函数定理

$\frac{1}{|\cos(\theta_1)-\cos(\theta_2)|} >= \frac{1}{2}$ ，所以 $\frac{d}{\lambda} = \frac{1}{\cos(\theta_1)-\cos(\theta_2)} < \frac{1}{2}$，空间混叠才不会发生。
