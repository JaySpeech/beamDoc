波束形成技术（三）：几种固定波束形成举例

发表于 2019-11-20 更新于 2019-11-19 分类于 语音增强 阅读次数： Valine：
本文字数： 2.1k 阅读时长 ≈ 2 分钟
介绍几种固定波束形成技术，Delay and Sum、加Chebywin窗、MaximumDF、Superdirective、Robust Superdirective、Null Steering、Subspace

介绍完固定波束形成和衡量它性能表现的指标之后，自然就过渡到几种常见的固定波束形成技术了。固定波束形成技术指的是阵列的加权系数不随空间声场变化而变化的阵列技术。但是我们可以控制不同的系数，以满足我们不同的性能要求（在开始设计阶段设计系数，一旦确定之后就钻石恒久远了）。比如，想要白噪声增益最大，或者想要指向性因数最大等等，但有个前提就是信号经过这些加权系数求和后不失真，总结概括起来就是：在约束条件下（信号不失真），最优化某个性能指标（$WNG 和 DF$）。所以波束形成问题就是优化问题。这个许多机器学习技术一样，在xxx条件下，优化xxx。所以在某种程度上，阵列问题就是机器学习问题。

后面，会以等间隔的线阵举例一些固定波束形成技术。在开始设计之前，聊一聊程序的设计模式。

程序结构设计

1.直线阵基类（Base类）。成员属性：阵列麦克风个数、间距、要设计的阵列指向、类名（也就是固定波束形成的类型）。

成员函数：计算单位时延、滤波器系数（抽象、后面具体的类重载）、导向矢量、扩散噪声场的协方差矩阵（名字待修正）、波束模式（单频、画图）、波束模式（几个典型频率、画图）、波束模式热力图等。

2.固定波束形成类（FixedBeamforming），继承上述的直线阵基类，成员函数有白噪声增益函数的计算及画图（随频率变化）、指向性因数的计算及画图。

3.具体的Fixedbeamforming。所有的类继承2中的固定波束形成类，只需要重载1中的滤波器系数的成员函数即可实现固定波束形成技术上述的所有功能。

4.一些基本的常量参数，考虑到可能的变更，在外层parameters中单独设计一个类控制，一些基本的矩阵操作，定义在外层的util中。以方便以后的维护及扩展。

白噪声增益达到了理论上的最大值的同时，$\mathcal{D}\left[\mathbf{h}_{\mathrm{DS}}\left(f, \cos \theta_{\mathrm{d}}\right)\right]>=1$ 的取值范围也使得阵列不会放大扩散噪声。

1.白噪声增益

只与麦克风个数有关的，随着麦克风个数增大，白噪声增益也变大。

2.指向性因数

指向性因数与麦克风个数、间距、阵列设计指向的角度都有关。麦克风个数越多，$DF$就越大。也可以看的到在低频的时候$DF$ 与 $WNG$ 相差较大，但高频的时候二者就很接近了。这是音频高频时候的扩散声场“近似”于一个白噪声场。
